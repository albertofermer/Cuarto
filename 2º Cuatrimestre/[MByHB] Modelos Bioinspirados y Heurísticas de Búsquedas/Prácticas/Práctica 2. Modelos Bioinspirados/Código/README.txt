Para ejecutar la práctica puede ejecutar el fichero 
main.py desde consola añadiendole los siguientes
parámetros:

---------------------------------------------------------
python main.py [nombre_algoritmo] (identificador_funcion)
---------------------------------------------------------

----------------------------------------------------------
Siendo:

- nombre_algoritmo: El nombre del algoritmo que quiera
ejecutar. A continuación se le muestra una lista de los
algoritmos disponibles:

['best', 'local', 'global']

- identificador_funcion: Solo será necesaria en caso de
utiliza el algoritmo de primero el mejor.
El 0 corresponderá con la función de Rosenbrock,
 mientras que el 1 la de Rastrigin.

-----------------------------------------------------------
