Para ejecutar la práctica puede ejecutar el fichero 
main.py desde consola añadiendole los siguientes
parámetros:

-----------------------------------------------------------------
python main.py [nombre_algoritmo] [semilla] [identificador_datos]
-----------------------------------------------------------------

----------------------------------------------------------
Siendo:

- nombre_algoritmo: El nombre del algoritmo que quiera
ejecutar. A continuación se le muestra una lista de los
algoritmos disponibles:

['basico', 'chc', 'multimodal']

- semilla: La semilla numérica que quiera probar con el
algoritmo seleccionado

- identificador_datos: 1 si quiere utilizar los datos aleatorios
y 0 si quiere utilizar los datos reales.
-----------------------------------------------------------
