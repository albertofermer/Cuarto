Para ejecutar la práctica puede ejecutar el fichero 
main.py desde consola añadiendole los siguientes
parámetros:

---------------------------------------------------
python main.py [nombre_algoritmo] [numero_problema]
---------------------------------------------------

----------------------------------------------------------
Siendo:

- nombre_algoritmo: El nombre del algoritmo que quiera
ejecutar. A continuación se le muestra una lista de los
algoritmos disponibles:

['greedy', 'random', 'best', 'first', 'vnd', 'es', 'tabu']

- numero_problema: El 0 corresponderá con el problema de
datos reales, mientras que el 1 al de datos aleatorios.

-----------------------------------------------------------

Una vez seleccionado el algoritmo se le indicará si debe
insertar más parámetros (semilla, granularidad...)

En caso de dejarlo vacío se establecerá un parámetro por
defecto.


==================================================================
Por otro lado, si quiere generar las gráficas tendrá que ejecutar
cada uno de los algoritmos por separado ya que el directorio
donde se almacenan las imágenes está en cada uno de los ficheros.
